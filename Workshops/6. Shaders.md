# Shader programming

Often we want to draw the player's attention to a specific object in our game world. One way to do this is adding a shine around an object.

Let's make a pixel shader to create this shine!


### C#
1. Add a new sprite effect in the MGCB editor.
2. Load the Sprite Effect and add it to the Spritebatch.Begin()

### HLSL
1. Read our current pixel value.
2. Is it transparant?
   - Calculate the distance to the sprite center
   - Make the pixel gold, with more transparancy the further from the center of the texture you are.
