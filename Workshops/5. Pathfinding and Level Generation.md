# Pathfinding

We have a unit that can only move orthagonally (up/down/left/right) at **S** that wants to go to **T**:

| 1  |   |   |   |   |   |   |
|---|---|---|---|---|---|---|
| 2  | **S** |   |   |   |   |   |
| 3  |   |   |   |   |   |   |
| 4  |   |   |   |   | **T** |   |
| 5  |   |   |   |   |   |   |
|   | A  | B  | C  | D  | E  | F  |

For the following heuristics, what is the next cell in the path:
  - Cartesian distance
  - Taxi-Cab distance

1. Calculate which cell comes first in the path.
2. Which comes next?
3. Which heuristic is more accurate? Does that mean it is better?

| 1  |   |   |   |   |   |   |
|---|---|---|---|---|---|---|
| 2  | **S** |   |   |   |   |   |
| 3  |   | X  | X  |  X |  X |   |
| 4  |   |   |   |   | **T** |   |
| 5  |   |   |   |   |   |   |
|   | A  | B  | C  | D  | E  | F  |

Tiles with an X are impassible.

Imagine we try to save some time by using the squared distance. 
1. What path will we find?
